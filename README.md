# entity-generator

Use this application to quickly load entity data into Conduce.

## prerequisites

This requires cmake and boost to build.  To install them on
osx run:

    brew install cmake
    brew install boost

## build/run

1. Clone repository
1. cd <new repo dir>
1. `mkdir build`
1. `cd build`
1. `cmake ../`
1. `make`
1. `./src/entity-generator/entity-generator`
